# Devin Review Checklist

## Code Review

- [ ] コードスタイルがプロジェクトの規約に従っている
- [ ] 不要なコメントやデバッグコードが残っていない
- [ ] 適切なエラーハンドリングが実装されている
- [ ] セキュリティ上の問題がない

## Testing

- [ ] ユニットテストが追加/更新されている
- [ ] テストが全て通る
- [ ] エッジケースが考慮されている

## Documentation

- [ ] 必要なドキュメントが更新されている
- [ ] API の変更がある場合、API ドキュメントが更新されている

## Performance

- [ ] パフォーマンスへの影響が考慮されている
- [ ] 不要な API コールやクエリがない

## Deployment

- [ ] 環境変数の追加/変更がある場合、適切に管理されている
- [ ] マイグレーションが必要な場合、適切に作成されている
- [ ] CI/CD パイプラインが通る
